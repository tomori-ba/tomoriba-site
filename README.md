# TOMORIBA Website

## ファイル構成

- `index.html`: サイト本体
- `style.css`: デザイン
- `script.js`: ヘッダーの動き
- `images/`: サイトで使う写真

## 更新方法

1. GitHubでファイルを編集またはアップロード
2. Commit changes
3. Vercelが自動で公開サイトを更新

## よく触る場所

- 開催日程：`index.html` の `SCHEDULE`
- SNS：`index.html` の `SNS`
- 写真：`images` フォルダと `index.html` / `style.css`
