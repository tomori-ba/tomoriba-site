## 更新方法

1. GitHubでファイルを編集またはアップロード
2. `Commit changes`
3. Vercelが自動で公開サイトを更新

## よく触る場所

- 開催日程：`index.html` の `Schedule`
- SNSリンク：`index.html` の `SNS`
- デザイン：`style.css`
- 写真：`images` フォルダ内の画像を同じ名前で差し替え
